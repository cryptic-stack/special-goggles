# web/app

Frontend app for `gnusocial-next`.

## Stack
- React + TypeScript + Vite
- React Router
- TanStack Query for server state
- Zustand for UI state
- Tailwind CSS
- lucide-react icons

## Run
```bash
npm install
npm run dev
```

## Current Milestone
- AppShell, navigation, sticky view headers
- UI Kit route
- Home/following/local/federated feeds with PostCard and skeletons
- Thread view with collapse/expand controls
- Composer with CW/visibility/language controls
- Search + Ctrl+K command palette + saved searches
- Notifications tabs and bulk controls
- Lists and bookmarks organization
- Settings for density, reduced motion, shortcuts, and role gates
