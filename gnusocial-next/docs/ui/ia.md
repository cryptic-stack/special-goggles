# UI Information Architecture

## Primary Navigation (Left Rail)
- Home
- Following
- Local
- Federated
- Explore
- Search
- Notifications
- Messages (optional toggle, hidden if disabled)
- Lists
- Bookmarks
- Profile (me)
- Settings
- Moderation (role-gated)
- Admin (role-gated)

## Global Elements
- Persistent compose button
- Instance switch and identity badge
- Quick search (Ctrl+K)
- Density toggle: Comfortable / Default / Compact

## View Structure Rules
- Each view has a sticky center header with title and controls
- Consistent action placement and spacing across feeds and threads
- Media reserves layout space to prevent content shift while loading
